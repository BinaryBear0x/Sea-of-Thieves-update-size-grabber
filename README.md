# Sea-of-Thieves-update-size-grabber

Bu repo, Steam News API üzerinden Sea of Thieves için yayınlanan son haber içeriğini çekip güncelleme notları ve indirme boyutu bilgisini ayıklamaya çalışan küçük bir PHP betiğidir.

## Ne Yapar

- Steam News API XML çıktısını okur
- başlık ve içerik alanlarını işler
- içerik içinden indirme boyutu bilgisini ayıklamaya çalışır

## Teknolojiler

- PHP
- DOMDocument
- Steam Web API
